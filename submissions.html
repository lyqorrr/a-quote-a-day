Submit a Quote

This page explains exactly how to submit your quote to A Quote A Day using GitHub✨

Step 1 · Create a New Quote File

Click the button at the bottom of this page. GitHub will automatically:

  • Create a copy (fork) of the project for you
  • Let you add a new quote file
  • Guide you to open a Pull Request

Step 2 · Filename Rules (Very Important)

Your quote is processed only if the filename starts with a date.

YYYY-MM-DD_name_department.md

Example:

2026-02-04_meenz_cyber-security.md
The date must be exactly in YYYY-MM-DD format (year-month-day, zero-padded).

Step 3 · Paste This Template Inside the File

Copy the template below and fill in all relevant fields. 

---
quote: 
author: 
contributor: 
department: 
source: Book / Film / Speech (optional)
tags: life, courage
about: Write why this quote matters to you
---
Empty or incomplete files may be ignored during processing.
